제어문
 - 조건제어문
   - if
      - if (조건) { 참 }
      - if (조건) { 참 } else { 거짓 }
      - if (조건) { 참 } else if (조건) { 참 } else if .....
   - switch ~ case
      - switch안의 자료형은 
        byte, short, char, int, String 형만 가능
      - case안에는 항상 상수값만 올 수 있다
 - 반복제어문 : 조건이 참이면 실행, 거짓이면 빠져나간다
   - for	: 수치에 의한 반복문
     - 단일 for : (초기값; 조건식; 증감식)
        - 초기값 : 시작이 되는 값, 초기값 지정 안할때도 있다
        - 조건식 : for문을 실행시킬 조건, 조건식도 지정 안할때가 있다
        - 증감식 : for문을 초기값에서 끝값으로 갈 수 있도록 하는 식	
     - 중첩 for
   - while	: 조건에 의한 반복문
     - while : (조건식)
     - do~while

※ print시 특수문자 : \
   - \n : 줄개행
   - \t : 탭만큼 이동(가장 가까운 탭의 위치로 이동)
